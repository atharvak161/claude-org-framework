↔️ Data Flow — Sidebar save → localStorage → Dashboard reads

fin_income → state.income
→ Overview: net pay, surplus, savings rate
→ Income tab: waterfall, scenario table
→ Net Worth: net pay in projection
→ OT predictions: base salary for forecast
fin_expenses → state.expenses
→ Overview: expense donut, surplus card
→ Expenses tab: category chart, monthly bar
→ Net Worth: monthly surplus calculation
→ OT forecast: expenses deducted per month
fin_investments → state.investments
→ Assets tab: portfolio donut, ULIP chart
→ Net Worth: all assets in timeline
★ Goals tab: Revolut adds to India trip
→ Overview: emergency fund gauge
fin_goals → state.goals
→ Goals tab: all three gauges
→ Overview: India trip gauge
→ OT forecast: start balance
★ Written back by OT confirm (savedGBP)
fin_ot_shifts → state.otShifts
→ OT tab: bar chart, savings forecast
→ Income tab: YTD OT total
→ Income page: monthly OT prediction
★ On confirm: writes to fin_goals.savedGBP
fin_debts → state.debts
→ Debts tab: amortisation chart
→ Net Worth: debt subtracted from NW
→ Overview: debt card
fin_tax_tracker → state.taxTracker
→ Tax tab: UK underpayment progress
→ Income page: extra tax deduction
fin_india_tax → state.indiaTax
→ Tax tab: India NRI summary cards
→ Tax tab: DTAA relief, ITR status
fin_settings → state.settings
→ All tabs: INR/GBP rate for £↔₹
→ Net Worth: pension growth, career date
→ Top bar: live rate display

✏️ Sidebar Pages — Click Edit · Fill Fields · Click Save

💰 Income
fin_income
  • Base salary (£/yr)
  • Average overtime (£/mo)
  • Hours per week
  • Pension employee % (of gross + OT)
  • Pension employer %
  • Tax code (e.g. 1034L)
  • Tax-free allowance (£/yr)
  • HMRC underpayment (£/mo)
  • Underpayment clears (date)
💸 Expenses
fin_expenses
  • Add / edit / delete expense items
  • Name, category, £/month, active
  • Scheduled changes (date + new £)
  • e.g. Rent drops Aug, EE ends Oct
🏦 Debts
fin_debts
  • SBI outstanding balance (₹)
  • Original principal (₹)
  • Interest rate (% p.a.)
  • Monthly EMI (₹)
  • Extra monthly payment (₹)
  • Loan start date
📈 Assets
fin_investments
  • Cash accounts (Revolut, balance, AER)
  • Pension (Standard Life, value)
  • ULIPs (SUD Life / PNB / Axis)
  • ISA (Stocks, Cash, Lifetime)
  • SIPP (value, contributions)
  • NPS Tier 1 & 2 (₹)
  • ELSS SIPs, PPF, Sovereign Gold Bonds
✈️ Goals
fin_goals
  • Emergency fund target (£)
  • Wealth target (£)
  • Target retirement age
  • India trip: saved, target, deadline
  • India trip budget breakdown
★ Dashboard adds Revolut to saved automatically
📉 Net Worth
fin_settings.nwProjection
  • Pension growth rate (% p.a.)
  • Career transition date
  • New salary after transition (£/yr)
⏱ OT Tracker
fin_ot_shifts · fin_ot_monthly_summary
  • Add / delete individual OT shifts
  • Date, gross amount (£), confirmed?
  • Mark month confirmed → actual net pay
  • What-if slider (display only)
★ Confirming writes savings to India trip goal
🧾 Tax Tracker
fin_tax_tracker · fin_india_tax
  • UK: underpayment total, monthly, dates
  • India: NRO interest, rental, dividends
  • India: TDS deducted (4 fields)
  • India: Section 80E (old regime only)
  • India: DTAA relief, Form 67, ITR status
⚙️ Preferences
fin_settings
  • Theme (dark / light)
  • INR/GBP rate (live fetch button)
  • Show INR equivalents toggle
  • Chart parameters
  • Data reset / export
