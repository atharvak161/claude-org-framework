# Cybersecurity Knowledge Base
# Owner: Knowledge Manager (coordinated with Security Architect)
# Last updated: 2026-06-01

## Purpose
This directory contains Atharva's personal cybersecurity notes, study materials,
CTF writeups, tool references, and any other security knowledge. All agents in the
Security department should read relevant files here before starting security tasks.

## Who uses this
- Security Architect — references for architecture decisions and threat modelling
- Penetration Tester — techniques, tools, and methodology references
- SAST/DAST Engineer — vulnerability references and testing approaches
- Compliance Auditor — security standards and regulatory references
- Portfolio Specialist — CTF writeups that can be published on portfolio site

## How to add files
Drop any .md, .txt, or .pdf file directly into this folder or a subfolder.
Suggested subfolders:
  notes/          — study notes and general theory
  tools/          — tool-specific references (Burp, Nessus, Metasploit etc.)
  ctf/            — CTF writeups and walkthroughs
  certifications/ — cert-specific notes (CEH, OSCP, CISM etc.)
  vulnerabilities/— specific vulnerability research
