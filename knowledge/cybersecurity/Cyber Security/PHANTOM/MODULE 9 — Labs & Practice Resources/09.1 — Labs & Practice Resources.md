# 09.1 — Labs & Practice Resources

## How to Build Practical Skill

Reading notes is passive. Skills are built by doing. This module maps out the best free and paid resources to practice every module covered in this curriculum, from beginner to advanced.

The fastest path to competency: **solve one machine per week, write a full report for each one.**

---

## Online Platforms

### HackTheBox (HTB)

https://www.hackthebox.com

The gold standard for practical pentesting practice. Real machines, realistic vulnerabilities, no hand-holding.

**Free tier:** Starting Point machines (guided) + active free machines  
**VIP (~$14/month):** Access to retired machines (thousands) + official walkthroughs

**Progression path:**

```
Starting Point (free)
  → Tier 0: Database, Preignition, Meow, Fawn (absolute basics)
  → Tier 1: Appointment, Sequel, Crocodile, Responder (real techniques)
  → Tier 2: Unified, Archetype, Oopsie (chained attacks)

Easy Machines
  → Blue (EternalBlue — Module 3)
  → Jerry (Tomcat JSP upload — Module 2)
  → Bashed (web → privesc — Modules 2+6)
  → Nibbles (web → privesc)
  → Netmon (PRTG RCE — research required)

Medium Machines
  → Forest (AD — Modules 4 specifically)
  → Active (Kerberoasting — Module 4)
  → Resolute (AD + privesc)
  → Bastard (Drupal RCE — Module 2)
  → Shocker (Shellshock — Module 2)

Hard Machines (when solid on Medium)
  → Reel (AD + phishing)
  → Sizzle (AD full chain)
  → Mantis (AD + MSSQL)
```

**HTB Academy:** https://academy.hackthebox.com — structured learning paths with labs. Covers most topics in this curriculum with hands-on exercises.

### TryHackMe (THM)

https://tryhackme.com

More guided than HTB. Better for complete beginners — rooms explain concepts as you go. Free tier available.

**Recommended learning paths:**
```
Pre-Security Path       → absolute foundations
SOC Level 1             → defensive fundamentals
Jr Penetration Tester   → covers all Module 1–6 topics
Advanced Exploitation   → Modules 3–4 depth
```

**Specific rooms to prioritise:**
```
Module 1 (Recon):        → "Passive Reconnaissance", "Active Reconnaissance"
Module 2 (Web):          → "OWASP Top 10", "Burp Suite", "Web Fundamentals"
Module 3 (Network):      → "Metasploit", "Network Services", "Hacking with PowerShell"
Module 4 (AD):           → "Active Directory Basics", "Attacktive Directory", "Post-Exploitation Basics"
Module 6 (PrivEsc):      → "Linux PrivEsc", "Windows PrivEsc"
Module 7 (Post-Expl):    → "Post-Exploitation Basics", "Lateral Movement and Pivoting"
```

https://portswigger.net/web-security

**Free. The best web application security training that exists.** Created by the makers of Burp Suite. Every topic in Module 2 has a dedicated lab track:

```
SQL injection           → 18 labs (beginner to blind to out-of-band)
XSS                     → 30 labs (reflected, stored, DOM)
CSRF                    → 12 labs
Clickjacking            → 5 labs
Access control          → 13 labs (IDOR, forced browsing)
Authentication          → 21 labs (brute force, MFA bypass, password reset)
Business logic          → 11 labs
File upload             → 7 labs
Server-side request forgery → 7 labs
XXE                     → 9 labs
Command injection       → 5 labs
SSTI                    → 7 labs
JWT attacks             → 8 labs
OAuth                   → 6 labs
HTTP request smuggling  → 22 labs
Web cache poisoning     → 12 labs
GraphQL                 → 5 labs
```

**Do every lab in a topic before moving to the next.** PortSwigger labs are the most effective web security practice available.

## Deliberately Vulnerable Applications

Run locally for focused practice on a specific topic.

### DVWA — Damn Vulnerable Web Application

https://github.com/digininja/DVWA

PHP/MySQL web app with configurable vulnerability levels (Low/Medium/High/Impossible).  
Covers: SQL injection, XSS, CSRF, file upload, command injection, IDOR, file inclusion.

```bash
# Docker
docker run --rm -it -p 80:80 vulnerables/web-dvwa
# Access: http://localhost/
# Default creds: admin / password
# Setup: http://localhost/setup.php → Create / Reset Database
```

### Metasploitable 2

Deliberately vulnerable Ubuntu VM. Covers network service exploitation (Module 3).

```bash
# Download from: https://sourceforge.net/projects/metasploitable/
# Run in VirtualBox on Host-Only network
# Default: msfadmin / msfadmin
```

### Metasploitable 3 (Windows)

Windows Server 2008 version with dozens of vulnerabilities — better for Windows attack practice.

```bash
# Build with Vagrant + VirtualBox
git clone https://github.com/rapid7/metasploitable3
cd metasploitable3
vagrant up win2k8
```

### VulnHub — OWASP WebGoat

Guided web app training with embedded hints. Good for step-by-step learning.

```bash
docker run -p 8080:8080 -p 9090:9090 webgoat/goatandwolf
# Access: http://localhost:8080/WebGoat
```

### bWAPP — Buggy Web Application

Over 100 web vulnerabilities in one application.

```bash
docker run -d -p 80:80 raesene/bwapp
```

## Reference Resources

### PayloadsAllTheThings

https://github.com/swisskyrepo/PayloadsAllTheThings

The single most useful offensive resource repository. Contains:
- Payload lists for every injection type (SQLi, XSS, SSTI, XXE, SSRF, LFI, etc.)
- Bypass techniques for every filter
- Methodology notes for each category
- Tool syntax and examples

**Bookmark this. Use it every time you test an injection point.**

### HackTricks

https://book.hacktricks.xyz

The offensive pentesting wiki. Contains detailed notes on:
- Every service attack (by port number)
- Linux and Windows privesc
- AD attacks
- Cloud attacks (AWS, Azure, GCP)
- Container escapes
- File type exploitation

**Treat it as your live reference during engagements.** Something weird on port 5985? Go to HackTricks and search "5985".

### GTFOBins

https://gtfobins.github.io  
Covered in [[06.3 — PrivEsc Tools Reference]]. Every Unix binary's abuse potential.

### LOLBAS

https://lolbas-project.github.io  
Covered in [[06.3 — PrivEsc Tools Reference]]. Every Windows binary's abuse potential.

### Exploit-DB / SearchSploit

```bash
# Local search (Kali)
searchsploit apache 2.4
searchsploit -x 12345    # examine an exploit
searchsploit -m 12345    # copy to current directory

# Online
https://www.exploit-db.com
```

### MITRE ATT&CK

https://attack.mitre.org

Map every technique you use to a TTP (Tactic, Technique, Procedure). Use this for report writing — "This attack corresponds to T1558.003 (Kerberoasting)" shows professional depth.

## Practice Methodology — How to Use These Resources Effectively

### The Right Way to Practice a Machine

```
1. Enumerate fully before exploiting anything
   (spend 30 mins on recon — resist the urge to run Metasploit immediately)

2. Research every version number you find
   (Apache 2.4.49? searchsploit it. MySQL 5.5? Check CVEs.)

3. Try manual exploitation before automated tools
   (Understand what's happening, then automate)

4. If stuck after 45 minutes → check the first hint only
   (Try to solve it with the hint before looking at a full walkthrough)

5. After rooting: write a report
   (Finding ID, CVSS, description, repro steps, impact, remediation)
   (This builds report writing muscle and reinforces learning)

6. Read the official writeup even if you solved it
   (There's almost always a technique you missed or a cleaner path)
```

### Structured Weekly Schedule (If Starting from Scratch)

```
Week 1–2:    PortSwigger SQLi track (all 18 labs)
Week 3–4:    PortSwigger XSS track (all 30 labs)
Week 5–6:    PortSwigger Auth track (all 21 labs)
Week 7:      HTB Starting Point Tier 0–1
Week 8:      HTB Starting Point Tier 2
Week 9–10:   HTB Easy machines (Blue, Jerry, Bashed)
Week 11–12:  PortSwigger Access Control + IDOR tracks
Week 13–14:  THM "Jr Penetration Tester" path
Week 15–16:  HTB Medium machines (Forest, Active — AD focus)
Week 17+:    One HTB machine per week + one PortSwigger track per week
```

## Certifications (Context for Your Career Path)

| Cert | Level | Content | Cost |
|---|---|---|---|
| eJPT (INE) | Beginner | Network + web basics | ~$200 |
| CEH (EC-Council) | Beginner-Mid | Theory-heavy, you already have this | — |
| PNPT (TCM Security) | Mid | Practical, AD focus, great value | ~$400 |
| OSCP (OffSec) | Mid-Senior | 24h practical exam, industry gold standard | ~$1,500 |
| BSCP (PortSwigger) | Mid-Senior | Web-only, Burp Suite mastery | ~$150 |
| CRTO (Zero-Point Security) | Senior | Red team, Cobalt Strike, AD | ~$450 |
| CRTE (AlteredSecurity) | Senior | Advanced AD | ~$250 |

**OSCP is the most recognised cert for junior penetration tester roles in the UK.** It validates practical skill, not exam memorisation. Target it after you're consistently solving HTB Medium machines.
